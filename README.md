# 🥘 Meal-Map

## 🚀 Vision

Meal-Map was created to remove the hassle of reading recipe instructions while cooking and manually setting timers.  
It turns saved recipes into a guided cooking timeline with automated timers and rich UI interactions.

---

## 📌 Key Features

### ⭐ Primary Features
- **Save Recipes** — Store your favorite recipes in a personal library.
- **Auto Timer** — Timers run automatically based on recipe steps.
- **Step-by-Step Cooking Flow** — Follow cooking steps with integrated timing.
- **Seamless Experience** — No need to switch between the recipe and clock apps.

### ✨ UI/UX Highlights
- Wipe animation for timer setup
- Animated clock icon while timers are active
- Dynamic Island-style floating timer UI
- Modern progress animation inspired by Nothing OS
